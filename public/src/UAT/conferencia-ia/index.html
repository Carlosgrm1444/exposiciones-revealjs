Nueva columna

Entonces como usar bien la IA?

Seguramente muchos estamos acostumbrados a copiar y pegar, dejar que resuelva todo, usarla para "salir del apuro" o depender de respuestas que no entendemos.

El uso correcto es usarla como tutor, pedirle rutas de aprendizaje, pedirle preguntas para definir mejores ideas, usarla para detectar huecos y para aprender más rápido.

No para pensar menos.
